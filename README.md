# Elevate Mindset AI 🚀

### Unlock Your Growth Potential! 🌱

**Elevate Mindset AI** is an AI-powered application designed to help users overcome challenges, learn from failures, and unlock their true potential. This tool promotes self-improvement through reflection, challenges, and achievements. 🌟

## Features ✨
- 💡 **AI-Powered Growth Assistant**: Provides motivation and guidance.
- 🔥 **Inspiring Quotes**: Daily motivational quotes to keep you inspired.
- 🎯 **Challenge Tracker**: Helps track your challenges and progress.
- 📊 **Progress Monitoring**: Visualizes your self-improvement journey.

## Tech Stack 🛠️
- **Frontend**: Next.js + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: MongoDB
- **AI Integration**: OpenAI API

## How to Run Locally 🖥️
```bash
# Clone this repository
git clone [https://github.com/WajahatAli3218664/GrowthMindsetAI-Q3Project]

# Navigate to project directory
cd elevate-mindset-ai

# Install dependencies
npm install

# Run the app
npm run dev
```
